## About the project

Simple web application to shorten the URLs.

## Techstack

Front-end:

- **React** _SPA_
- **react-router**

Back-end:

- **TypeScript** with **NodeJS** runtime
- **NestJS** for handling HTTP requests and API routing

Database:

- **Redis** for storage

This app also implements custom hashing algorithm based on the sha256.
